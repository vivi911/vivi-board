## ⚠️ 工作規範（每次任務都必須遵守）

### 完成任務後必須做以下三件事：

1. 更新本檔案（CLAUDE.md），把改了什麼記錄進去
2. commit + push 到 GitHub
3. 輸出變更摘要，格式如下：

---
任務：（任務名稱）
動了哪些檔案：
  - 檔案名稱（說明改了什麼）
可能影響的功能：
測試方式：
有沒有推 git：✅ / ❌
---

### 需要 code review 時，額外輸出：

---
⚠️ Code Review 請確認：
- [ ] 有沒有 hardcode 的 API key 或敏感資料
- [ ] 有沒有 error handling
- [ ] 邏輯有沒有明顯漏洞
- [ ] 有沒有應該更新但沒更新的 CLAUDE.md
---
